README
El codigo se ejecuta con python3 de la siguiente forma
python3 compilador.py test1.bas

Los tres primeros tests tienen el siguiente nombre
test1.bas
test2.bas
test3.bas

Los tests que deben mandar error de parsing tienen el siguiente nombre
test1f.bas
test2f.bas
test3f.bas

Los tres ultimos tests que deben pasar a pesar de tener scopes incorrectos son los siguientes
test1n.bas
test2n.bas
test3n.bas
